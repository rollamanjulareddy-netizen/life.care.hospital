# 🏥 LifeCare Hospital Website

## 📌 Project Description
LifeCare Hospital is a responsive hospital website developed using HTML, CSS, and JavaScript. The website provides information about the hospital, medical services, doctors, facilities, health tips, gallery, emergency support, and contact details.

## ✨ Features
- Responsive Home Page
- About Us Page
- Medical Services
- Doctors Information
- Hospital Facilities
- Gallery
- Health Tips
- Contact Page
- Emergency Services
- Responsive Navigation Bar

## 💻 Technologies Used
- HTML5
- CSS3
- JavaScript
